Question 3.1: Create a plot to visualize the relationship between DJs’ ages and their ratings.

Solution:

# complete the code to generate the plot

ggplot(data=playlist_data,mapping=aes(x=Age,y=Rating)) +
 geom_point() +
 labs(x="Age",y="Rating",
title="Age vs Rating",
caption="liddat lor")
